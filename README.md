# Skill 库数据看板

> 团队 Skill 库每日自动更新的数据看板，由悟空每天 17:50 自动生成并推送。

## 查看看板

🔗 **在线访问**：[https://lyl-0722.github.io/skill-dashboard/](https://lyl-0722.github.io/skill-dashboard/)

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | Skill 库数据看板页面（每日自动覆盖更新） |

## 看板内容

- **KPI 指标**：Skill 总数 / 累计调用量 / Commit 活跃度 / 贡献者数 / 仓库体积
- **图表**：分类 Skill 数量环形图 / Commit 活跃度柱状图 / 调用量 Top Skill 条形图 / 版本分布饼图
- **Skill 清单**：全量 Skill 列表，含分类、版本号、调用量、评分

## 数据来源

数据从 GitHub 仓库 [lyl-0722/team-skills-demo](https://github.com/lyl-0722/team-skills-demo) 实时拉取。
